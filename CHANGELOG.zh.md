# 更新日志

本项目的所有重要变更都将记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2026-03-04

### 新增
- Extension Manager 首次发布
- 模式管理功能，内置默认、极简、开发三种模式
- 扩展启用/禁用功能
- 导入/导出扩展配置
- 操作历史记录追踪
- 深色模式支持（浅色/深色/跟随系统）
- 多语言支持（中文和英文）
- 扩展搜索功能
- 批量启用/禁用操作
- 拖拽排序扩展列表
