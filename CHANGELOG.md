# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-03-04

### Added
- Initial release of Extension Manager
- Mode management feature with Default, Minimal, and Development presets
- Extension enable/disable functionality
- Import/Export extension configurations
- Operation history tracking
- Dark mode support (Light/Dark/System)
- Multi-language support (Chinese and English)
- Search extensions functionality
- Batch enable/disable operations
- Drag-and-drop extension sorting
